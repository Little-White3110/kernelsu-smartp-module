使用说明
KernelSU 模块的基本使用方法

KernelSU 是一个内核级的权限管理模块，允许您以 root 权限运行应用程序。

功能特点

  • check_circle
    内核级权限管理
  • check_circle
    支持 Magisk 模块
  • check_circle
    安全的权限控制
  • check_circle
    模块化设计

使用步骤

  1. 安装 KernelSU 模块
  2. 重启设备
  3. 打开 KernelSU 应用
  4. 授予应用 root 权限
