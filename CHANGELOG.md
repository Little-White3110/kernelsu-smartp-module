# KernelSU SmartP 模块 - 发布说明

## v1.0 (2026-04-13)

### 新增功能
- 实现了智能性能优化功能
- 集成了完整的备份和恢复系统
- 提供了现代化的 Web 界面管理
- 支持自动和手动备份
- 支持备份管理和恢复功能

### 技术实现
- 基于 KernelSU 框架开发
- 采用模块化设计，易于维护和扩展
- 提供了完整的启动流程和服务管理
- 支持 KernelSU 10600+ 版本

### 安装说明
1. 确保设备已安装 KernelSU 10600+ 版本
2. 将模块 zip 文件通过 KernelSU 应用安装
3. 重启设备
4. 模块将自动初始化并创建初始备份

### 卸载说明
1. 通过 KernelSU 应用卸载模块
2. 模块会自动清理备份文件和临时文件
3. 重启设备
